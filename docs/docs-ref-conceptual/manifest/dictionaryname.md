# <a name="dictionaryname-element"></a>DictionaryName-Element

Gibt den Namen dieses Wörterbuchs an.

**Add-In-Typ:** Aufgabenbereich

## <a name="syntax"></a>Syntax

```XML
<DictionaryName DefaultValue="string" />
```
