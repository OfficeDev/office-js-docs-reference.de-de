# <a name="citationtext-element"></a>CitationText-Element

Gibt die Zitatbausteine für dieses Wörterbuch an.

**Add-In-Typ:** Inhalt, Aufgabenbereich, E-Mail

## <a name="syntax"></a>Syntax

```XML
<CitationText DefaultValue="string" />
```
