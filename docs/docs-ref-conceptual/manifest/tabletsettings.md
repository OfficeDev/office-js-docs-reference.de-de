# <a name="tabletsettings-element"></a>TabletSettings-Element

Gibt Steuereinstellungen an, die gelten, wenn Ihr E-Mail-Add-In auf einem Tablet verwendet wird.

**Add-In-Typ: ** E-Mail-Nachrichten

## <a name="syntax"></a>Syntax

```XML
<TabletSettings>
   ...
</TabletSettings>
```
