# <a name="queryuri-element"></a>QueryUri-Element

Gibt die URL des Endpunkts für den Wörterbuch-Abfragedienst an.

**Add-In-Typ:** Aufgabenbereich

## <a name="syntax"></a>Syntax

```XML
<QueryUri DefaultValue="string" />
```
