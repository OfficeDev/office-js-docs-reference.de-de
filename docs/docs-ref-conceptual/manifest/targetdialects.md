# <a name="targetdialects-element"></a>TargetDialects-Element

Definiert die Regionalsprache oder Sprachen, die von diesem Wörterbuch unterstützt werden.

**Add-In-Typ:** Aufgabenbereich

## <a name="syntax"></a>Syntax

```XML
<TargetDialects>
   ...
</TargetDialects>
```
