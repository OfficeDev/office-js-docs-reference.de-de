# <a name="dictionaryhomepage-element"></a>DictionaryHomePage-Element

Gibt die URL der Startseite für das Wörterbuch an.

**Add-In-Typ:** Aufgabenbereich

## <a name="syntax"></a>Syntax

```XML
<DictionaryHomePage DefaultValue="URL" />
```
