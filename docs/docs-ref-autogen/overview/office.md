# <a name="office-javascript-api-reference"></a>Office JavaScript-API-Referenz

Die JavaScript-API für Office ermöglicht Ihnen das Erstellen von Webanwendungen, die Interaktion mit den Objektmodellen in Office-hostanwendungen. Verwenden Sie in diesem Abschnitt erfahren Sie mehr über die Klassen, Methoden und andere Typen zum Erstellen von Office-Add-ins zur Verfügung.

- [Excel](https://docs.microsoft.com/javascript/api/excel?view=office-js)
- [OneNote](https://docs.microsoft.com/javascript/api/onenote?view=office-js)
- [Outlook](https://docs.microsoft.com/javascript/api/outlook?view=office-js)
- [Visio](https://docs.microsoft.com/javascript/api/visio?view=office-js)
- [Word](https://docs.microsoft.com/javascript/api/word?view=office-js)
- [Freigegebene API](https://docs.microsoft.com/javascript/api/office?view=office-js)
